Figure Skating Scoring

In figure skating, because it is an international sport, the International Judging System (IJS) is used to calculate figure skaters' scores. Through this system, each figure skating element is given a base value (BV) of points and then additional points for the Grade of Execution (GOE) are added on depending on how well a skater performed an element. Below is a list of common elements, and their base values.

Discipline Element Base Value
Singles/ Pairs Double Toe Loop 1.04
Singles/ Pairs Double Salchow 1.04
Singles/ Pairs Double Loop 1.36
Singles/ Pairs Double Flip 1.44
Singles/ Pairs Double Lutz 1.68
Singles/ Pairs Double Axel 2.64
Singles/ Pairs Triple Toe Loop 3.36
Singles/ Pairs Triple Salchow 3.44
Singles/ Pairs Triple Loop 3.92
Singles/ Pairs Triple Flip 4.24
